# Privacy Policy — Volt EV Calculator

Last updated: March 28, 2026

Volt EV Calculator is a browser extension that displays Expected Value (EV) information for packs on getvolt.gg.

## Data Collection

This extension does not collect, store, or transmit any personal data. No user information is sent to any external server.

## How It Works

The extension reads marketplace data that is already loaded by your browser when you visit getvolt.gg. All calculations are performed locally on your device.

## Third-Party Services

This extension loads the Inter font from Google Fonts via CSS for UI styling purposes. No remote JavaScript is fetched or executed.
